# How to Use This Index

Whenever a new experiment is created:

1. Add the experiment folder under `/experiments`
2. Write a short `notes.md`
3. Add one line to this table summarizing the insight

The key insight should answer:

**"What did I learn from this experiment?"**

Example:

| Experiment | Date | Area | Key Insight |
|------------|------|------|-------------|
| aws-lambda-test | 2026-04 | AWS | Lambda simplifies event-driven workloads |
| terraform-modules | 2026-04 | IaC | Modules dramatically reduce Terraform duplication |
